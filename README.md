dc-caspar
=========

Behringer CMD DC-1 CasparCG Controller.


Use [this](http://www.behringer.com/EN/Products/CMD-DC-1.aspx) 99$ Midi controller to execute predefined CasparCG macros with visual feedback

Requires / based on
-------------------
- Python 3
- PyQT5
- python-rtmidi
